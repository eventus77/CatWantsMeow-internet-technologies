Данный тренинг посвящён актуальным подходам, технологиям и инструментальным средствам разработки на Javascript. Посещая занятия данного курса, вы узнаете о последних тенденциях и трендах в JavaScript (с фокусом на ООП), познакомитесь с HTML, CSS и HTTP; научитесь выполнять юнит-тестирование клиентского кода; познакомитесь с архитектурой приложений и структурой типовых проектов, а также получите представление об инструментах современного Javacript-разработчик.

Требования к кандидатам:

  1. Знание основ программирования, ООП и алгоритмизации;

  2. Технический английский (на уровне уверенного чтения технических документов).

Темы, которые будут затронуты на тренинге:

  1. Современный JavaScript. HTML5 APIs.

  2. HTML. От отдельных тегов до Semantic Web.

  3. CSS3. Подходы к верстке страницы. Анимация. Responsive Web Design. CSS препроцессоры LESS и Stylus.

  4. Разработка через тестирование в JavaScript. Юнит-тестирование клиентского кода. Автоматизированное функциональное тестирование.

  5. Архитектура приложения. Single Page Application. AngularJS.

  6. Структура проекта. Модульность. Управление зависимостями и ресурсами. Grunt. AMD, RequireJS.

  7. Инструменты современного веб-разработчика. Sublime Text. Web Storm. Google Chrome Developer Tools. Бонус: Git и Git Flow.
